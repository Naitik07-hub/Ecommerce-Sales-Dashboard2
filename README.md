# Ecommerce-Sales-Dashboard2
# 📊 E-commerce Sales Dashboard (SQL + Excel)

## Overview

This project analyzes e-commerce sales data using SQL and visualizes insights in Excel.

## Tools Used

* SQL (Data extraction & aggregation)
* Excel (Dashboard & visualization)

## Key Insights

* USA generates highest revenue (~3.3M)
* Top 3 customers contribute ~40%
* Peak sales observed in November

## Files

* `sql/queries.sql` → all SQL queries
* `excel/dashboard.xlsx` → final dashboard
* `images/dashboard.png` → preview
